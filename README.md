### To Do List
<p>
<img src="https://github.com/vivekdamani/ToDo-Android-App/blob/master/to-do%20icon.jpeg" width="400px"><br>This application is to keep a track of things you need to do and tick them once they are done. You can even easily edit and delete thing to do in your list.
<p>


### Prerequisites

Android studio 4.0 or above and a Network connection

## Getting Started

1. Clone this Repository
2. Open Android studio
3. Open an existing project in android studio
4. Build (using the hammer icon) and install the app.

## Features
1. Easy to add multiple tasks in your list.
2. Easy to edit and delete tasks.
3. Can tick done once your task is completed.
